各平台上超强系列输入法的安装

发表于2021-08-23|更新于2024-05-17|超强快码超强两笔超强音形各平台的安装
|阅读量:

前言

超强系列输入法已可在目前主流平台(Android/Ios/Windows/Linux/Mac)上完美运行，依托 Rime 的开源跨平台特性及其强大功能，使超强输入法如虎添翼。限于技术和精力，某些平台上只做了 Rime 的安装包，期待有更多志同道合的朋友加入制作队伍，让用户有更多选择，让更多人使用。

安装包下载地址见这里。

RIME 是一个通用的输入法引擎，它支持几乎所有的常见平台：

  • 在 Windows 上是小狼毫
  • 在 MAC 上是鼠须管
  • 在 Linux 上是 ibus-rime 或 fcitx-rime 或 fcitx5-rime
  • 在手机安卓平台上是同文输入法
  • 在 IOS（苹果手机）上是 iRIME 或 仓输入法

为什么我们会使用 rime 平台？一言以蔽之：它是一款极为强大的开源跨平台免费输入法引擎。
因为是一个通用输入法引擎，所以我们可以使用一份配置来支持所有这些引擎。
RIME 的官网在此

本文介绍已经打包好的 Win/Mac/Linux 平台的超强系列 rime 的安装与使用。
与 Rime 配置语法都是相互关联的一样，本教程所涉及到的知识点或关键点也都是相互关联的，只看某一部分可能并不能解决问题。

在 Linux/Mac/Windows 上安装 Rime 版超强系列输入法

功能概述

  1. 按 F4 鍵或組合鍵 Ctrl+` 喚出輸入方案選單（` 鍵常見於 1 的左方）在超强快码、超强两笔、超强音形间切换（某些软件快捷键可能冲突，两种快捷键都试一下，或换个能打字的软件再试）。不清楚自己用的是哪种超强方案去这里找答案

  2. 按 「 i 」 键再打全拼，或不打声母只打两笔的形码部分，可反查编码。如超强快码「你」字可打 ini 或 ihek 。

  3. 键入 help 显示帮助菜单。

  4. 超强快码、超强音形使用分号 「 ; 」 打临时英文，超强两笔用引号 「 ‘ 」。另外可使用分号 「 ; 」 来选二重，用引号 「 ‘ 」 选三重。

  5. 无码加词、调频，详见这里

  6. 更多功能详见这里

  7. 重要：若想改配置请在用户文件夹下操作 ，这里有各系统下 rime 用户文件夹位置的详细介绍。

  8. Linux 下由于情况特殊功能分基础版和高级版，详见安装包内说明。

安装

下载超强系列官方网盘里的文件名带有 小狼毫 字样的安装包。极少数情况下安装后若打不了中文，需要右键以管理员权限运行你安装目录下的这个文件 WeaselServer.exe ,如果你没有更改安装目录的话一般在：C:\Program Files (x86)\rime\cqkm\ 下,这个是算法服务，如果你每次开机都要运行一次此服务可以使用小狼毫助手或中州韵助手的进程守护功能保活。也有可能是你装了杀毒软件拦截了，自行添加白名单。

苹果电脑系统也叫 Mac ，可以下载超强系列官方网盘里 Mac 目录下的 鼠须管 或 小企鹅 ，此小企鹅是带 rime 插件的，完美支持 rime 。参照安装包内说明安装或网盘下的视频教程。

linux 系统下安装 rime（中州韵） 可以下载超强系列官方网盘里 linux 目录下的文件名带有 linux-rime 的，此安装包集成了多种 linux 发行版的安装脚本，并分为基础版和高级版，用户按需安装。

  1. 在正式安装之前，我们要先确认自己系统里正在使用的是哪种输入法框架，在 linux 系统里，常见且好用的输入法框架目前有三种，即 ibus/fcitx/fcitx5 ，不过大多数 linux 发行版已自带安装好，我们要分辨一下自己使用的是哪种，才能进行下一步操作。下图是这三种框架的设置截图，新手可以以此作参考来判定自己目前使用的哪种框架：
  2. 确定自己使用的是哪种输入法框架之后再安装与之对应的 rime 中州韵引擎，如确定你使用的是 ibus 框架，则安装 ibus-rime ,使用的是 fcitx ，则安装 fcitx-rime ,使用的是 fcitx5 ，则安装 fcitx5-rime 。有些发行版自带有软件商店之类的（比如 uos），可以在商店里搜索“中州韵”或 “rime” ，再直接安装即可（这里最好还是再对比一下是哪种输入法框架）。若没有商店的，可以在终端下安装，仓库包名一般为 ibus-rime,fcitx-rime,fcitx5-rime，如你现在使用的输入法框架是 fcitx5，则终端安装 fcitx5-rime 包。Rime官网有详细说明。已安装的跳过。

统信 uos 特别说明：
统信 uos 20 专业版 1060 政务授权版（控制中心——系统信息处可看到），要在应用商店里搜索”中州韵“或”rime“，安装 中州韵输入法，若应用商店里搜索不到，不要在终端安装中州韵（以后可能会修复此问题吧）。

  1. 正式开始安装：

先确认一下 rime（中州韵） 是否安装成功，完成安装后重启并在输入法设置里添加「中州韵」，添加中州韵后，在打字时在任务栏可以将输入法切换为中州韵就说明安装好了。好了，到这一步说明你已经安装好对应的 rime 了，下面我们要把超强系列的码表导入进去。将下载的超强系列安装包解压后，先看附带安装说明，里面有几个安装脚本，我们要选择一个脚本来安装，各脚本作用是：

  • install-1.sh              基础版，只要能装 rime(中州韵) 就可用（基础功能，够用，建议优先安装）
  • install-2.sh              高级版
  • deepin23-install.sh       deepin23 专用(deepin20用install-1.sh)
  • ubuntu.sh               ubuntu 专用

基础版：（install-1.sh）

  • 按 F4 或 CTRL+~ 在 超强快码、超强两笔、超强音形 之间切换，空格确认，不清楚自己用的是哪种超强方案去这里找答案
  • 加词详见这里
  • 方案的配置文件是 *.schema.yaml，具体参考用户文件夹下 cqkm_cm.schema.yaml ，不想折腾的不用管。
  • 一切修改都要保存后重新部署才能生效

高级版：（install-2.sh ，需 lua 支持）

  • 含基础功能
  • 键入 help 出现帮助菜单，下翻有更多
  • 关于拆分提示，只显示超强快码单字全码的第2、3、4码对应的字根
  • 这里有部分高级功能示例

选择好脚本后，正式开始超强系列的码表导入。在解压后的安装包目录下空白处点右键选择“在此处打开终端”，在终端输入你选择好自己要用的安装脚本再回车，如 sh install-1.sh

完成安装后重启并在输入法设置里添加「中州韵」。添加中州韵后，在打字时在任务栏将输入法切换为中州韵后，就可使用超强两笔系列方案打字了。按 F4 或 CTRL+~ 在 超强快码、超强两笔、超强音形 之间切换，空格确认，不清楚自己用的是哪种超强方案去这里找答案

  1. 高级功能的一点说明：
    fcitx 无法使用高级功能。kde fcitx5 下若要使用高级功能，再进行下列操作：

    • fcitx5 配置 – 配置附加组件 – 经典用户界面 – 字体（不是下面那个 菜单字体）– 选择字体为 cqkmcf ,然后重新启动 fcitx5 就可以了，不行就重启系统。
    • fcitx5 要装 lua 组件方可使用 lua 功能，Arch 系包名为 fcitx5-lua .您如果是 Debian 用户，请用 Debian Testing ，由于 Debian 11 上 fcitx5 的版本太低，官方仓库里缺少 librime-plugin-lua 这个基本组件，无法正常使用本档配置文件。
    • deepin 23 下执行 deepin23-install.sh 后再执行上面第1条设置字体后即可完美使用。deepin 20 无法使用高级功能，除非把输入法框架改为 fcitx5 。
    • ubuntu 执行完 ubuntu.sh 后，重启或注销系统一次后，在「区域与语言」中添加「汉语-rime中州韵」，再在程序中搜索「extensions」，就是那个「扩展」，在 ibus-tweaker 中设输入法字体为「cqkmcf」。可以参照这里五笔的视频安装及设置教程，里面中州韵助手部分不用理会：https://www.bilibili.com/video/BV1eW4y1h71i/
    • 如果您是其它发行版，请自行查询 librime 版本是否已经包含了 librime-plugin-lua 这个基本组件。另，若安装基础版后正常使用，而高级版无法输入任何汉字可能是 lua 组件版本过低。

  2. 总结：

  • 安装ibus/fcitx/fcitx5 任意框架(一般都略过)
  • 安装与之对应 Rime 引擎

原生码表是网盘 linux 目录下的文件名带有 table 的，如 Fcitx-table 或 Fcitx5-table 。
原生码表仅作备用，即 Rime 版安装不了的，可以用原生码表“救急”。原生码表没有 rime 那么多功能，不易维护自己的词库，不过胜在安装方便。
原生码表只做了两种，fcitx/fcitx5 ，由于超强系列两笔与五笔等不同，有一些特殊，ibus 原生码表总是有一些问题，就不做了。根据自己电脑上已安装的输入法框架选择，区分这三种框架见前面介绍安装 “ Linux_Rime 版(推荐) ” 的第一条。如你的电脑上本身已安装好 fcitx，则安装 Fcitx-table 码表。下面以安装 fcitx 原生码表为例。其它操作方法一样。

在解压后的安装包目录下空白处点右键选择“在此处打开终端”，在终端输入安装命令再回车，如 sh install.sh

安装完成后照上图所示添加超强系列方案，上图示例的是在 fcitx 下添加超强方案，fcitx5 添加方法大同小异，自行摸索。不清楚自己用的是哪种超强方案去这里找答案

注：

  1. 输入上面命令或手动复制还是不能用？输入法设置里添加选项没有超强？
    可能没有安装 table包，以 fcitx 为例，ubuntu/debian/uos终端安装：sudo apt install fcitx-table 。manjro系统:sudo pacman -S fcitx5-table-extra,其它发行版自行搜索安装，仓库都有。装好后再重启输入法。具体对应为：

    • fcitx: fcitx-table
    • fcitx5: fcitx5-table-extra

  2. 如何加词？

    • fcitx/fcitx5
      先把你要加的词打出来，再按fcitx5默认ctrl+8即可根据提示加词。ctrl+7为删词，ctrl+6为调频。要批量加大量词需要编译码表。

  3. 如何用分号和引号选二重和三重？
    fictx 配置里，选择你要修改的方案配置，如选超强快码，后面有个按钮，出来个面板，默认只有几项，把对话框往下稍微拖大一点其它选项就出来了，在“选择第二第三候选词”里输入 ;'

  4. 如何临时拼音？
    由于 fcitx/fcitx5 的配置问题，不能用 i 键，默认为 [ 打临时拼音。
