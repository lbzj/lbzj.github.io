前言

超强系列输入法已可在目前主流平台(Android/Ios/Windows/Linux/Mac)上完美运行，依托 Rime 的开源跨平台特性及其强大功能，使超强输入法如虎添翼。限于技术和精力，某些平台上只做了 Rime 的安装包，期待有更多志同道合的朋友加入制作队伍，让用户有更多选择，让更多人使用。

在 Linux/Mac/Windows 上安装 Rime 版超强系列输入法

与 Rime 配置语法都是相互关联的一样，本教程所涉及到的知识点或关键点也都是相互关联的，只看某一部分可能并不能解决问题。

RIME 是一个通用的输入法引擎，它支持几乎所有的常见平台：

  • 在 Windows 上是小狼毫
  • 在 MAC 上是鼠须管
  • 在 Linux 上是 ibus-rime 或 fcitx-rime 或 fcitx5-rime
  • 在手机安卓平台上是同文输入法
  • 在 IOS（苹果手机）上是 iRIME

为什么我们会使用 rime 平台？一言以蔽之：它是一款极为强大的跨平台免费输入法引擎。
因为是一个通用输入法引擎，所以我们可以使用一份配置来支持所有这些引擎。
RIME 的官网在此

本文介绍已经打包好的 Win/Mac/Linux 平台的超强系列 rime 的安装与使用。

功能概述

  1. 按 F4 鍵或組合鍵 Ctrl+` 喚出輸入方案選單（` 鍵常見於 1 的左方）在超强快码、超强两笔、超强音形间切换（某些软件快捷键可能冲突，两种快捷键都试一下，或换个能打字的软件再试）

  2. 按 「 i 」 键可拼音反查编码，不会读的字亦可用其引导。按 「 ~ 」 键可编码反查拼音

  3. 键入 help 显示 lua 帮助菜单，可直接输出当前时间日期等等，功能较多，自行测试

  4. 超强快码、超强音形使用分号 「 ; 」 打临时英文，超强两笔用引号 「 ‘ 」。另外可使用分号 「 ; 」 来选二重，用引号 「 ‘ 」 选三重。

  5. 无码加词、调频。（加入个人词库详见用户文件夹下面 cq**.src.dict.yaml文件）

  6. 精准造词。按 ` 键引导并分隔输入串。如：欲造「精准造词」，输入jy`zuh`zrv`ci后依次选字即可。 如果已存在词组「精准」和「造词」，也可输入jyzu'zrci。所造的词不易备份，有临时词时可以用，想所造的词长期保留建议还是加到用户文件夹下 cq**.src.dict.yaml 中。
    想删除已造词在打出来时光标移到该词条按下Shift+Delete或Control+Delete（蘋果鍵盤用 Shift+Fn+Delete）。

若想改配置请在用户文件夹下操作
各系统(PC端)对应 rime 程序文件夹和用户文件夹路径在：

右键任务栏“中”字图标

  • 用户文件夹：~/Library/Rime
  • 程序文件夹：/Library/Input\ Methods/Squirrel.app/Contents/SharedSupport
  • 程序文件夹：/usr/share/rime-data/
  • 用户文件夹：
    • fcitx : ~/.config/fcitx/rime
    • ibus: ~/.config/ibus/rime
    • fcitx5: ~/.local/share/fcitx5/rime
      这里有三个用户文件夹，分别对应三个输入法框架，只看自己用的哪个就行了，如你使用的 fcitx 就只看 ~/.config/fcitx/rime 这里就行了。

安装

极少数情况下安装后若打不了中文，需要右键以管理员权限运行你安装目录下的这个文件 WeaselServer.exe ,如果你没有更改安装目录的话一般在：C:\Program Files (x86)\rime-cqkm\ ,这个是算法服务，如果你每次开机都要运行一次此服务可以加入计划任务，或使用小狼毫助手保活。也有可能是你装了杀毒软件拦截了，自行添加白名单。

直装版想添加自定义词组将程序文件夹下cq*.src.dict.yaml复制到用户文件夹下再操作

在 linux 下安装超强系列输入法的视频教程在此

再说一次，原生码表仅作备用，即 Rime 版安装不了的，可以用原生码表“救急”。原生码表没有 rime 那么多功能，基本上仅能打字，加词调频之类的功能太弱。
原生码表只做了三种，ibus/fcitx/fcitx5 ，根据自己电脑上已安装的框架选择，区分这三种框架见上面介绍安装 Rime 版的第一条。如你的电脑上本身已安装好 fcitx，则安装 fcitx 码表。下面以安装 fcitx 原生码表为例。其它操作方法一样。

注：

  1. 输入上面命令或手动复制还是不能用？输入法设置里添加选项没有超强？
    可能没有安装 table，以 fcitx 为例，ubuntu/debian/uos终端安装：sudo apt install fcitx-table。manjro:sudo pacman -S fcitx5-table-extra,其它发行版自行搜索安装，仓库都有。装好后再重启输入法。具体对应为：

    • ibus : ibus-table
    • fcitx: fcitx-table
    • fcitx5: fcitx5-table-extra

  2. 如何加词？

    • fcitx/fcitx5
      先把你要加的词打出来，再按fcitx5默认ctrl+8即可根据提示加词。ctrl+7为删词，ctrl+6为调频。要批量加大量词需要编译码表。
    • ibus
      每输入一个字都用Ctrl+Num进行选择，直到词组输入完毕，然后按空格提交
      还有一种就是在normal commit状态下，先把词组输入完，但是不空格确认，此时按左shift，再空格就可以添加

  3. 如何用分号和引号选二重和三重？
    fictx 配置里，选择你要修改的方案配置，如选超强快码，后面有个按钮，出来个面板，默认只有几项，把对话框往下稍微拖大一点其它选项就出来了，在“选择第二第三候选词”里输入 ;'

  4. 如何临时拼音？
    由于 fcitx/fcitx5 的配置问题，不能用 i 键，默认为 [ 打临时拼音。ibus 为 右shift
