超强快码的规则



一、十三个字根

  1. 月辶日艹氵木口亻钅小土彳扌。

    口诀：月走日草水木口，人金小土双人手。

  2. 字根整体取码，不能拆分为笔画。

  3. 如字根有其他笔画穿过，则不再视为字根。

二、键盘分区及键位排列

  1. 按从上到下，从左到右分为撇区、竖区、折区、点区、横区。

  2. 键位对称排列，从中间到两边分别为撇、横、折、点、竖。

  3. 单笔画归于同名分区，置于每区的中指键。

  4. 横竖置于 M 键，折竖置于 N 键。

    口诀：分区撇竖折点横，键位撇横折点竖；

    单笔归区中指取，横竖 M 折竖 N

三、独体字编码规则

取码顺序 第一码 第二码 第三码 第四码
取码要素 拼音首字母 第一二笔 末笔 末码

如： 雨 YMII = Y（首音）＋M（一丨）＋I（丶）＋I（末码）

四、合体字编码规则

  1. 后半为合体结构
取码顺序 第一码 第二码 第三码 第四码
取码要素 拼音首字母 前半一二笔 后半首部第一二笔 后半次部第一二笔

如： 撕 SUMT = S首音）＋U（扌）＋M（一丨）＋T（ノノ）

  1. 后半为独体结构
取码顺序 第一码 第二码 第三码 第四码
取码要素 拼音首字母 前半一二笔 后半一二笔 后半末笔

如： 铺 PJMI = P（首音）＋J（钅）＋M（一丨）＋I（丶）

五、词组编码规则

  1. 二字词：取每字前两码。

    如： 教程 JMCR

  2. 三字词：取第一字前两码和后两字第一码。

    如： 输入法 SKRF

  3. 四字词：取每字第一码。

    如： 超强快码 CQKM

  4. 多字词：取前三字和末字第一码。

    如： 中华人民共和国 ZHRG

六、补码规则

  1. 不足四码的字，补末码至四码。

    如： 丁 DMMM = D（首音）＋M（一丨）＋M（末码）＋M（末码）

  2. 打足四码而出现重码的字词，补EIOUVW键选第2～7个重码

    如： 共享 GMXUE 海南 HSNMI

超强快码进阶

一、疑难字拆分原则

  1. 半包围结构字的拆分构件优先，再按构件笔顺取码。

    如： 式 SKMK = S（首音）＋K（弋）＋M（ㄒ）＋K（一）

  2. 多级结构字的拆分取大优先，最大构件整体取码。

    如： 幕 MADD = M（首音）＋A（莫）＋D（冂）＋D（丨）

  3. 有明显界线的字视为合体字，并以该界线拆分。

    如： 丽 LKDD = L（首音）＋K（一）＋D（冂）＋D（冂）

  4. 夹有“冖”的字以此为分界线，且“冖”不编码。

    如： 营 YAGG = Y（首音）＋A（艹）＋G（口）＋G（口）

  5. 夹在结构中的独立单笔，以此为分界线，且该笔画不编码。

    如： 修 XHET = X（首音）＋H（亻）＋E（夂）＋T（彡）

  6. 合体字取完第二半的笔画，不能再取第一半的笔画。

    如： 困 KDFF = K（首音）＋D（冂）＋F（木）＋F（末码）

  7. 可分出类似字根的结构，按字根编码，视为合体字。

    如： 百 BHNN = B（首音）＋H（丆）＋N（日）＋N（末码）

  8. 一个字被另一个结构天然分开，视为两个独立结构。

    如： 街 JILJ  = J（首音）＋I（彳）＋L（圭）＋J（亍）

二、全形输入与拼音输入

  1. 不会读的字可选择全形方式，方法是“i＋单字全形”。

    如： 瘿 IUDB

  2. 不会写的字可选择拼音方式，方法是“i＋单字拼音”。

    如： 睿 IRUI

三、难点解析

  1. “冒”字上部并非“曰”，故编码为 MDDK。

  2. “敝”字前两笔为点撇，故编码为 BYRI。

  3. “直”字前两笔为横竖，故编码为 ZMDK。

  4. “出”字前两笔为折竖，故编码为 CNDD。

  5. “万”字前两笔为横折，故编码为 WKEE。

  6. “脊”字前两笔为点横，故编码为 JUCC。

  7. “凹”字前两笔为竖折，故编码为 ADKK。

  8. “凸”字前两笔为竖横，故编码为 TFKK。

  9. “尴”字前两笔为横撇，故编码为 GHAD。

  10. “〇”字笔画为折，故编码为 LCCC。

四、扩展功能

注：此为多多输入法平台功能

  1. 可用 ; ’上屏第二、三选项

  2. 向前翻页快捷键：↑

  3. 向后翻页快捷键：↓

  4. 用户造词快捷键：Ctrl++

  5. 简入繁出快捷键：Ctrl+J

  6. 编码查询快捷键：Ctrl+K

  7. 大写金额快捷键：`+数字

  8. 当前日期快捷命令：date

  9. 当前时刻快捷命令：time

  10. 显示 Unicode 字符，须安装超大字符集支持包。

超强快码输入法简介

  1. 由付东升发明的汉字键盘音形码输入法。

  2. 每字全码均为四码，不足四码用末码补足。

  3. 首码是声母，二、三、四码为笔画或字根。

  4. 充分统计音码和笔画，合理布局，指法顺手。

  5. 键位布局规律性强，字根数量少，易学易记。

  6. 采用全分两半规则，具有极强的直观性。

  7. 覆盖全部GBK字符集及《通用规范汉字表》。

  8. 精选词库，具有超大容量，超低重码的特点。
